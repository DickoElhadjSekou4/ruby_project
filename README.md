# Ruby Project – Boucles et Méthodes

# Description

Ce projet est un entraînement en Ruby sur les méthodes et les boucles.  
Chaque programme est contenu dans un fichier du dossier `lib` et possède une méthode `perform` permettant de l’exécuter facilement.

## Structure du projet
## Détail des fichiers

# 00_hello.rb
Demande le prénom de l'utilisateur et affiche un message de bienvenue.

# 01_pyramids.rb
Génère :
- une demi-pyramide
- une pyramide complète
- une pyramide en losange

# 02_password.rb
Simulation d’un système de mot de passe :
- signup
- login
- écran secret

# 03_stairway.rb
Jeu d’escalier :
- Le joueur monte ou descend selon un lancer de dé.
- Une méthode permet de calculer le nombre moyen de tours pour gagner.
